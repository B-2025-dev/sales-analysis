# Sales Analysis Project

This project is a **Python-based sales analysis tool** that reads sales data from a CSV file, calculates total sales, and visualizes the results with bar and pie charts.

## Features

- Load and analyze sales data from `sales.csv`
- Calculate total sales per product
- Visualize sales:
  - **Bar chart**: Total sales per product
- Supports monthly sales analysis
